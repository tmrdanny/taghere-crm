# 스크린샷 촬영 가이드

매뉴얼에 필요한 스크린샷 목록입니다. 각 스크린샷에서 빨간색 박스로 강조할 영역을 안내합니다.

---

## 촬영 도구 추천

### macOS
- **CleanShot X** (유료): 빨간 박스, 번호, 화살표 추가 가능
- **Skitch** (무료): 기본 주석 기능
- **스크린샷 앱 + 미리보기**: 기본 도구로 도형 추가

### 빨간 박스 스타일
- **색상**: #FF0000 또는 #E53E3E
- **두께**: 2-3px
- **모서리**: 둥근 모서리 (radius 4-8px)
- **번호 표시**: 흰색 원 안에 빨간 숫자 (①②③④⑤)

## 스크린샷 편집 팁

### 빨간 박스 추가 방법 (Figma 사용 시)
1. 스크린샷을 Figma에 삽입
2. Rectangle 도구로 강조할 영역 위에 사각형 그리기
3. Fill: None, Stroke: #E53E3E, Stroke Width: 3px
4. Corner Radius: 8px
5. 번호가 필요한 경우 Ellipse로 원 그리고 텍스트 추가

### 빨간 박스 추가 방법 (미리보기 앱 사용 시)
1. 스크린샷 파일 더블클릭하여 미리보기로 열기
2. 마크업 도구 클릭 (연필 아이콘)
3. 도형 > 사각형 선택
4. 색상 빨간색 선택, 테두리만 설정
5. 저장
